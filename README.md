# Personal Interview Prep App

A small, single-user interview prep app for one learner. Its job is to make daily study easier: plan the day, track revision, run coding and Q&A practice, and keep weak topics visible until interviews start in July.

## Current phase

This repository is intentionally in the **Establish** phase: requirements, feature scope, architecture, data flow, diagrams, tech stack, resource sources, and July learning plan are documented before app implementation begins.

## Primary outcome

The app should help the learner:

1. Explain React, Next.js, TypeScript, React Native, FastAPI, and Flask principles in interview Q&A.
2. Build practical code with those technologies under timed constraints.
3. Explain trade-offs and how to apply technologies to real product problems.
4. Track daily progress toward interview readiness within July.
